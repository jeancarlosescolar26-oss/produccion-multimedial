Responda las siguientes preguntas:
1. ¿Qué es HTML y para qué se utiliza?
R/: HTML es el lenguaje de marcado de hipertexto utilizado para estructurar contenido en la web, definiendo elementos como texto, imágenes y enlaces. Se usa principalmente para crear páginas web y aplicaciones que los navegadores interpretan visualmente.
2. ¿Cuál es la función de las etiquetas html, head y body ?
R/: html es la etiqueta raíz que envuelve todo el documento. head contiene metadatos como el título y enlaces a estilos, sin mostrarse al usuario; body incluye el contenido visible de la página.
3. ¿Qué diferencia hay entre una etiqueta y un atributo?
R/: Una etiqueta define el tipo de elemento (ej. (p) para párrafo), mientras que un atributo añade información extra dentro de la etiqueta de apertura (ej. href="url" en a) para configurar propiedades
4. ¿Para qué se usa la etiqueta title?
R/:La etiqueta title se usa dentro de head para definir el título de la página, que aparece en la pestaña del navegador y en resultados de búsqueda.
5. Mencione tres etiquetas semánticas de HTML5
R/: tres etiquetas semánticas de HTML5 son header (para cabeceras), footer (para pies de página) y article (para contenido independiente).
